# Embedded Web Portfolio

A personal portfolio website with an "Embedded Systems" aesthetic, built with Vanilla HTML/CSS/JS and Vite.

## Features
- **Boot Sequence**: Simulated system startup animation.
- **Theme**: Dark mode, PCB background patterns, CRT scanlines, and terminal typography.
- **Sections**: Hero, System Specs (About), Runtime Logs (Experience), Loaded Modules (Projects), I/O Ports (Contact).
- **Responsive**: Works on desktop and mobile devices.

## Prerequisites
- **Node.js**: Required to run the development server. [Download Node.js](https://nodejs.org/)

## Setup & Running
1. Open a terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the URL shown in the terminal (usually `http://localhost:5173`).

## Customization
- **Content**: Edit `index.html` to update your bio, experience, and projects.
- **Styles**: Edit `src/style.css` to change colors or effects.
- **Logic**: Edit `src/main.js` to adjust the boot sequence timing.
